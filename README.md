# EPUB Studio

EPUB Studio は、  
**フォルダ構成をもとに EPUB ファイルを生成する macOS アプリ**です。

画像・テキストを含むフォルダを入力として、
Apple Books でそのまま読める EPUB を生成することを目的としています。

---

## 🎯 目的

- フォルダに整理した素材から EPUB を作りたい
- 専用エディタや複雑な設定を使わずに EPUB を生成したい
- Apple Books での閲覧を前提とした EPUB を手軽に作りたい

EPUB Studio は  
**「EPUB 制作の敷居を下げる」**ことを目標に設計されています。

---

## ✨ 主な機能（v0.1.0）

### 📁 フォルダベース EPUB 生成
- 指定したフォルダ構成をもとに EPUB を生成
- フォルダ名・ファイル名を章構成として利用

### 🖼 画像対応
- 画像を含む EPUB の生成に対応
- 画像サイズの調整・最適化を実施

### 📖 Apple Books 対応
- Apple Books での表示を前提とした EPUB を生成
- Finder から Books ライブラリへの登録を確認済み

### 🧭 シンプルな操作フロー
- 入力フォルダを選択
- 必要な情報を入力
- EPUB を生成

---

## 🧩 設計方針

- **フォルダ構造 = 本の構造**
- EPUB の内部構造を意識せずに使えること
- 最小限の UI で迷わず操作できること

将来的な拡張を見据え、
ロジックと UI を分離した構成で実装されています。

---

## ⚠️ 制限事項（v0.1.0）

- EPUB の詳細なスタイル指定は未対応
- メタデータ（詳細な著者情報など）の拡張は今後対応予定
- 大規模フォルダ構成での最適化は未対応
